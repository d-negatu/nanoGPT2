# Tokenizer Module

Documentation for the tokenizer module in nanoGPT2.

## Overview

The tokenizer module handles text preprocessing and token encoding/decoding.

## Key Features

- BPE (Byte Pair Encoding) tokenization
- Token vocabulary management
- Encoding text to token IDs
- Decoding token IDs back to text
- Special token handling
